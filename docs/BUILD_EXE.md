# Сборка exe для Windows (без Python на сервере)

Один exe: **tg_scan.exe** (бот + сверка AD в 01:00 + реестр в 07:00). На сервере Python не нужен.

## Где собирать

Сборку нужно делать **на Windows**. На машине для сборки: Python и зависимости проекта.

## Шаг 1: Подготовка

```cmd
pip install -r requirements.txt
pip install pyinstaller
```

## Шаг 2: Сборка

```cmd
build_all.bat
```

Готовый файл: **dist\tg_scan.exe**

## Шаг 3: Что положить на сервер

Скопируйте в одну папку (например, `C:\tg_scan\`):

| Файл / папка | Откуда | Назначение |
|--------------|--------|------------|
| **tg_scan.exe** | `dist\` | Бот + планировщик (сверка AD 01:00, реестр 07:00) |
| **config.ini** | создать из config.example.ini | Настройки: токен, A-Tracker, SMTP, группы |
| **data\** | создать папку | Реестры (.xls/.xlsx); ad_export.json для синхронизации AD |

## Шаг 4: Запуск

**По умолчанию** — бот + планировщик (сверка AD каждый день в 01:00, реестр в 07:00):

```cmd
tg_scan.exe
```

**Только проверка реестра** (один раз, выход):

```cmd
tg_scan.exe --registry
tg_scan.exe --registry data
```

Рабочая папка при запуске — папка с exe и config.ini.

## Важно

- Скопируйте **config.example.ini** в **config.ini**, заполните секреты (токен, пароли).
- config.ini не должен попадать в git.
