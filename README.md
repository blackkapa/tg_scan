# Инвентаризация техники (A-Tracker + веб)

Проект связывает **A-Tracker** с корпоративной почтой и веб-интерфейсом: авторизация по коду, список активов, инвентаризация по QR, заявки (перемещение, добавление техники, несоответствие).

## Быстрый старт

1. Установите зависимости:

   ```bash
   pip install -r requirements.txt
   ```

2. Скопируйте `config.example.ini` в `config.ini` и заполните URL A-Tracker, SMTP, секцию `[web]` и т.д.

3. Запуск веб-интерфейса:

   ```bash
   uvicorn front_site.app:app --host 0.0.0.0 --port 8000
   ```

   Подробнее: `front_site/README.md`.
